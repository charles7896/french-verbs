# Entraîneur de verbes français

A simple, offline French verb conjugation trainer. Choose the verbs and tenses you
want to practise and take an interactive quiz that checks each answer and tracks
your score.

**Live site:** https://YOURNAME.github.io/french-verbs/

## Features
- 200 common French verbs (regular -ER, -IR, -RE and the main irregulars)
- 11 tenses: présent, imparfait, futur simple, conditionnel présent, impératif,
  passé composé, participe présent, participe passé, futur proche, subjonctif
  présent, passé simple
- One-click "Semaines 33–34" preset
- Built-in accent keyboard, instant correction, and an end-of-quiz review
- 100% offline, no accounts, no data collection

## Running locally
It's a static site — just open `index.html` in any browser.

## Hosting
Served as static files via GitHub Pages (Settings → Pages → Deploy from branch → `main` / root).
